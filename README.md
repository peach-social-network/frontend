## Цель

Реализовать социальную сеть - Modern Network.

---

## Функционал

- Видео / аудио звонки
- Видео / аудио сообщения, сообщения, временные сообщения, сообщения на будущее, сообщения без уведомлений
- Иметь возможность переключения для режима работы
    1. Обычный: В этом режиме у пользователя будет доступен полный функционал приложения.
    2. Рабочий: Доступны только чаты. Включая звонки
    3. Кастом: Здесь пользователь сможет сам настраивать то, что ему нравится. Посты и рабочий чат, видео и т.д.

---
